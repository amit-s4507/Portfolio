## 🔄 Migration Guide

### From v0.1.0 to v1.0.0

1. **Update dependencies**:

   ```bash
   npm install
   ```

2. **Environment setup**:

   - Copy `env.example` to `.env.local`
   - Configure EmailJS credentials

3. **Build and test**:
   ```bash
   npm run build
   npm run dev
   ```
